## 기능 요건

| ID | 이름 | 설명 | 우선순위 |
|----|------|------|----------|
| F-01 | 일감 생성·수정·삭제 | 제목(필수)·설명·마감일·태그를 입력해 일감을 만든다. 저장 후 즉시 보드에 반영된다. | 상 |
| F-02 | 담당자 배정 | 일감에 팀원 1명을 배정한다. 아바타로 시각화하며 미배정 상태도 허용한다. | 상 |
| F-03 | 상태 관리 | 할 일 → 진행 중 → 검토 중 → 완료 4단계를 카드 이동 또는 드롭다운으로 변경한다. 단계를 앞·뒤로 모두 이동할 수 있다. | 상 |
| F-04 | 우선순위·마감일 경고 | 낮음·보통·높음·긴급 4단계 우선순위를 카드 왼쪽 컬러 바로 표시한다. 마감 D-2 이내는 주황, 초과는 빨간색으로 강조한다. | 중 |
| F-05 | 담당자·우선순위 필터 | 아바타 클릭으로 담당자를, 뱃지 클릭으로 우선순위를 필터링한다. 텍스트 검색(제목·설명)과 중복 적용되며 초기화 버튼으로 한 번에 해제한다. | 중 |

## 비기능 요건

### 성능

| ID | 항목 | 기준 |
|----|------|------|
| NF-P1 | 초기 로드 | Lighthouse Performance 점수 **90점 이상** (모바일 기준) |
| NF-P2 | 상태 변경 반응 | 카드 상태 변경 후 UI 반영까지 **100 ms 이내** (localStorage 기준) |

### 보안

| ID | 항목 | 기준 |
|----|------|------|
| NF-S1 | 입력값 길이 제한 | 제목 최대 **200자**, 설명 최대 **2,000자** — 초과 입력 시 저장 불가 |
| NF-S2 | XSS 방지 | 사용자 입력을 innerHTML 없이 React 텍스트 노드로만 렌더링, dangerouslySetInnerHTML 미사용 |

### 가용성

| ID | 항목 | 기준 |
|----|------|------|
| NF-A1 | 오프라인 복원 | 브라우저 새로고침 후 데이터 손실 **0건** (localStorage 영속) |
| NF-A2 | 크로스 브라우저 | Chrome·Edge·Safari 최신 2버전에서 핵심 기능 **100% 동작** |

### 접근성

| ID | 항목 | 기준 |
|----|------|------|

| NF-AC2 | 색상 대비 | WCAG 2.1 AA 기준 — 텍스트 대비율 **4.5:1 이상** |
